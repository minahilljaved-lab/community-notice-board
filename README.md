# Community Notice Board App 📋

A fully functional desktop application built with Java Swing as a semester project.

## Features
- User registration and login with secure file storage
- Create, view, and delete community posts
- Categories — Announcements, Events, Lost & Found, Buy/Sell, General
- Search and filter posts by keyword or category
- Image attachments on posts
- Event reminder notifications

## How to Run
Make sure Java is installed on your computer, then:
java -jar CommunityBoard.jar

## Built With
- Java (Swing GUI)
- File I/O for data persistence
- Object-Oriented Programming principles
  - Encapsulation, Inheritance, Polymorphism, Abstraction, Association

## Project Structure
- User.java / Post.java — data models
- UserStorage.java / PostStorage.java — file handling
- LoginFrame / RegisterFrame / DashboardFrame / CreatePostFrame — UI screens
- NotificationService.java — event reminder system
- Main.java — application entry point
